#数组

#取样器（纹理）

取样器（sample）用于访问纹理，有两种基本的取样器类型：

  • sampler2D
  • samplerCube

取样器变量只能是uniform变量。

#流程控制

if、if-else、过多的if条件会降低着色器的执行速度；

没有switch语句；

for循环只允许又一个循环变量，且该变量只能是int或float类型；

循环表达式只能是以下形式：i++、i--、i+=常量表达式或者i-=常量表达式；

#函数

参数限定字

类别规则
in向函数中传入值，并且可以修改
const in向函数中传入值，但是不能修改
out在函数中被赋值，并被传出
inout传入函数，同时在函数中被赋值，并被传出
无将一个值传给函数，和in一样

例如将颜色值转转换为明度的方法：

void luma(in vec3 color, out float brightness) {
  brightness = 0.2126 * color.r + 0.7162 * color.g + 0.0722 * color.b;
}

#内置函数

#存储限定字

const

定义常量；

attribute

只能出现在顶点着色器中，只能被声明为全局变量，被用来表示逐顶点的信息；

被修饰的变量只能是以下类型：float、vec2、vec3、vec4、mat2、mat3、mat4；

varying

必须是全局变量，用于从顶点着色器向片元着色器传输数据；

被修饰的变量只能是以下类型：float、vec2、vec3、vec4、mat2、mat3、mat4；

#精度限定字

  • highp 高精度

  • mediump 中精度

  • lowp 低精度

#ifdef GL_ES
	precision mediump float;
#endif

#预处理指令

预处理指令用来在真正编译之前对代码进行预处理

#if 条件表达式
	//条件为真，执行这里
#endif
#ifdef 某宏
	//如果定义了某宏，执行这里
#endif
#ifndef 某宏
	//如果没有定义某宏，执行这里
#endif

#ifdef可以配合#else一起使用

定义宏： #define 宏名 宏内容

#define NUM 100
#ifdef NUM == 100
  //如果宏NUM为100，执行这里
#else
  //执行这里
#endif

#version number 来指定着色器使用的GLSL ES版本，可以接受的版本包括100（GLSL ES 1.00）和101（GLSL ES 1.01）默认100。该指令只能在着色器顶部。
