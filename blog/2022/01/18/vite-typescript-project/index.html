Vite打包TypeScript库工程

2022-01-18 by 墨鸿 - 阅读需要1分钟

Vite的出现极大的降低了打包ts库工程的复杂度，同时也使得开发过程中的demo开发更加方便。

初始化工程

通过以下命令先初始化一个vue-ts工程：

npm init vite@latest my-vue-app -- --template vue-ts

然后删除vue相关的依赖和插件。

生成.d.ts声明文件

通过rollup插件rollup-plugin-typescript2可生成声明文件，在vite.config.ts如下配置：

import typescript2 from 'rollup-plugin-typescript2'

export default defineConfig({
  plugins: [
    {
      ...typescript2(),
      apply: 'build'
    }
  ]
})

还需要在tsconfig.json中设置以下字段：

{
  "compilerOptions": {
    // 是否生成声明文件
    "declaration": true,
  },
}

示例文件

直接在根目录下创建examples文件夹，文件夹内创建html文件，通过script引用src/index.ts文件即可：

<script type="module" src="/src/index.ts"></script>

可以在浏览器里面直接访问这个html文件。
